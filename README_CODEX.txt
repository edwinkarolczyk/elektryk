Domowy Elektryk v1.2.0 — nowość: GRAFICZNA ROZDZIELNICA

• Paleta: MAIN (FR), RCD (30mA), MCB (B16), RCBO, SPD, BLANK.
• Dodawanie: wybierz typ po prawej → kliknij w siatkę rozdzielnicy (24px pola).
• Przeciąganie: kliknij moduł i puść w docelowym miejscu (snap do siatki).
• Szerokość (poles): wbudowana wg typu (np. RCD=2). Kolumny 18, wiersze 8 (seed).
• Etykieta: „Zmień etykietę…” (po ID modułu).
• Usuwanie: „Usuń zaznaczony moduł” (po ID).
• Przypinanie do obwodu: wybierz obwód w liście → „Przypisz do zazn. obwodu”, podaj ID modułu.
  – dla MCB/RCBO kolor modułu = kolor obwodu; etykieta uzupełniana na podstawie obwodu.

Szybki start:
1) Utwórz pliki wg # FILE: i uruchom `python start.py`.
2) Zakładka „Rozdzielnica” → kliknij w siatkę, żeby dodać aparaty.
3) Zakładka „Plan / Mapa” nadal działa: elementy, kable w kolorach obwodów.

Pomysły na v1.3+:
• kolizje (blokowanie nakładania się modułów),
• info-overlay z ID przy najechaniu,
• edytor rozmiaru (poles) i koloru dla dowolnego modułu,
• auto-generacja listy aparatów (BOM) i eksport do CSV/PDF,
• przypinanie wielu MCB pod jedną RCD (wizualne „mostkowanie”).
